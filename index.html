О чем говорят цветы?

"Сколько бы мы ни говорили о пустоте жизни, иногда достаточно лишь одного цветка, чтобы нас разубедить."

Значение разных цветов
Миндаль - надежда
алое - горе
Абрикос - искушение
Туя - вечная дружба
Ясень - величие
Земляника - простота
Барбарис - вспыльчивость
Базилик - ненависть
Черника - предательство
Ромашка - милосердие
Георгин - элегантность
Нарцисс - безответная любовь
однако не только вид цветов показывает на их значение, есть не сколько других категорий, которые могут кардинально изменить значение цветка, например такие как:
  1. по цветам
    2. красный
    белый
    желтый
  2. по культурам
    3. европейская
    китайская
